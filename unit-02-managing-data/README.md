# Unit 2 — Managing Data in ML

The data-preparation unit — the largest by notebook count. Covers the full path from raw data to a clean, model-ready dataset: filtering, class balancing, type handling, encoding categorical variables, exploratory visualization, summary statistics, and handling missing values and outliers.

**Contents:** 13 notebooks — 1 lab, 1 assignment, 11 practice notebooks.
