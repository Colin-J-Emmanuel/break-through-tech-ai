# Unit 1 — ML in a Nutshell

An orientation to the machine learning life cycle and the Python toolkit used throughout the program. The lab covers business understanding and problem formulation; the practice notebooks build fluency with Jupyter, NumPy, and pandas.

**Contents:** 5 notebooks — 1 lab, 4 practice notebooks.

| Notebook | Type | What it demonstrates |
|---|---|---|
| [`Lab-01_ML-Life-Cycle-Business-Understanding.ipynb`](./labs/Lab-01_ML-Life-Cycle-Business-Understanding.ipynb) | Lab / Capstone | End-to-end walkthrough of the ML life cycle, focusing on business understanding and framing a problem before any modeling. |
| [`Practice_Intro-to-Jupyter-Notebooks.ipynb`](./practice/Practice_Intro-to-Jupyter-Notebooks.ipynb) | Practice | Getting started with the Jupyter notebook interface, cells, and execution. |
| [`Practice_Jupyter-Notebook-Basics.ipynb`](./practice/Practice_Jupyter-Notebook-Basics.ipynb) | Practice | Hands-on practice with core notebook workflows. |
| [`Practice_NumPy-Vectorization-Broadcasting.ipynb`](./practice/Practice_NumPy-Vectorization-Broadcasting.ipynb) | Practice | NumPy fundamentals: vectorized operations and broadcasting for efficient numerical computing. |
| [`Practice_Pandas-Creating-Viewing-Data.ipynb`](./practice/Practice_Pandas-Creating-Viewing-Data.ipynb) | Practice | Creating, inspecting, and viewing data with pandas DataFrames. |
